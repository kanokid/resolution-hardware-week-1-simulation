This is a circuit simulation I made in Falstad that has three LEDs. From left to right, they go off after each other. The circuit uses a decoupling capacitor to keep energy after the voltage source is stopped by the switch and uses a resistor to slow down the current flow. And there are diodes to prevent the current from leaking the other way which would ruin the circuit flow.
